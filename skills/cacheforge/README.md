# CacheForge

Primary entrypoint skill for CacheForge workflows in OpenClaw.

This skill bootstraps companion skills and routes tasks to:

1. `cacheforge-setup` (onboarding)
2. `cacheforge-ops` (account operations)
3. `cacheforge-stats` (usage telemetry)

## Install

```bash
clawhub install cacheforge
```
